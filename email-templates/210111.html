  • Under Advent of Code samlade 50 personer tillsammans in 1276 stjärnor! Bra jobbat alla. Dessutom lyckades hela 11 personer få alla 50 stjärnor. Extra grattis till Jonathan Wase som var snabbast att lösa och kammade hem största vinsten! Om du fått mersmak av problemlösningen börjar vårens IMPA nu på måndag (18e januari). På meetupen dagen efter (19e januari) kommer vi -- om intresset finns -- gå igenom hur systemen (UVA, IMPA och Kattis) fungerar och hur du kommer igång. Kom förbi vår Discord-server om det låter intressant!
  • Nästa Game Jam är Global Game Jam (GGJ) 29e-31a januari. GGJ är som namnet antyder ett globalt Game Jam med arrangörer över hela världen och LiTHe kods Game Jam-grupp LiU Game Jam håller i eventet i Linköping. Kolla in deras hemsida för mer information och deras Discord-server där eventet kommer hållas.
  • Den 3 februari kommer vi hålla en introducerande git-workshop för fundersamma studenter. git är ett välanvänt versionshanteringssystem som du som student kan använda för att enklare programmera med andra, både i mindre labbar och större projekt som ett kandidatarbete. Om du redan har hört om git och använt det tidigare är det också ett utmärkt tillfälle att finslipa hur du gör t.ex. en rebase, bisect eller blame. Mer information finns på vår hemsida.
  • I vår ska en ny styrelse väljas in och redan nu är ansökan öppen! Att vara med i styrelsen är ett ypperligt tillfälle att göra något extra för föreningen och lära känna nytt folk. Det är inte heller lika tidskrävande som många tror. Styrelsen har skrivit små texter på hemsidan där de presenterar sina poster och vad de innebär; kika in där om det låter intressant! Anmälan görs via formulär.

  • On the 3rd of February we will hold an introductory git-workshop for curious and/or frustrated students. Git is a widely used version control system that you as a student can use to collaborate with others easier, both in smaller labs and larger projects like a Bachelor's thesis. If you've already heard about git and used it before it is also an excellent oppurtunity to polish your rebases, bisects and blames. More information can be found on our website.
